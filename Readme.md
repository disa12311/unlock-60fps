# Unlock 60FPS NO ROOT

## Installation Command

```sh
sh /sdcard/60FPS/Execute.sh
```
or
```sh
sh /storage/emulated/0/60FPS/Execute.sh
```

## Uninstallation Command

```sh
sh /sdcard/60FPS/Delete.sh
```
or
```sh
sh /storage/emulated/0/60FPS/Delete.sh
```
